# Zyklone Bot

Hello, I'm Zyklone, a poker dealer Discord bot.

## What I Can Do

- Be A Poker Dealer
- (more features coming soon)

## Setup

Install all dependencies:
    
    npm install

Create env file with a Discord API Token:

    touch .env

    // inside .env file:
    TOKEN='add your token here'

Run your script of choice:

    // dev
    npm run start:dev

    // production
    npm run start:prod

    // debug
    npm run start:debug
